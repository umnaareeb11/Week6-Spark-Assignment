# Week 6 - Spark Assignment

This is my Week 6 assignment for the Data Engineering course. 
In this assignment I learned how Apache Spark works and how 
to process data using PySpark.

## What I did in this assignment

- Understood Spark Architecture (Driver, Cluster Manager, Executors)
- Learned how Lazy Evaluation works in Spark
- Read CSV and Parquet files using PySpark
- Performed filtering and selecting of columns
- Renamed columns and changed data types
- Added new columns with calculations
- Built a complete data pipeline (read → filter → write)
- Learned the difference between Transformations and Actions
- Understood concepts like DAG, Predicate Pushdown and Fault Tolerance

## Tools Used

- Python
- PySpark 4.0.3
- Google Colab

## Files

- Week6_Spark_Assignment.ipynb → main assignment notebook

## What I learned

This assignment helped me understand how Spark processes 
large datasets efficiently using distributed computing. 
I also learned why Parquet is better than CSV for big 
data and how to handle null values in a dataset.
